Question 6

Cos’è il this in JavaScript?".

Risposta:

Utilizzando "This" manipoliamo l'oggetto corrente. "This" è una parola chiave che viene riferita ad un oggetto specifico. Il valore di "This" non può essere cambiato in quanto non è una variabile. Possiamo utilizzare "This" dentro un metodo che si trova al'interno di un oggetto. In questo caso "This" farà riferimento al nome dell'oggetto. Nel esempo sotto "This" si riferisce all'oggetto "car".

const car = {
brend: "Ford",
model: "Puma",
maxSpeed: 200,
gearbox: manual,
reliseDate: 2019{
fullName: function(){
return this.brend + " " + this.model
}
};
